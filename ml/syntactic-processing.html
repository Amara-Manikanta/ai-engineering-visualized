NLP Application

Syntactic Processing

Analyzing the grammatical structure of sentences using Part-of-Speech (POS) tagging and Parsing algorithms.

Core Techniques

Lexicon & Rule-Based POS Tagging

Using dictionaries (lexicons) and hand-written grammatical rules to assign POS tags (Noun, Verb, Adjective). Often uses corpora like the WSJ (Wall Street Journal).

HMM-based POS Tagging

Hidden Markov Models (HMM) probabilistically determine the most likely sequence of tags based on transition and emission probabilities (e.g., using the Viterbi algorithm).

Parsing

Constructing a syntax tree (Constituency Parsing) or a dependency graph (Dependency Parsing) to understand the exact relationships between words.

Information Extraction

Extracting structured entities (Named Entity Recognition) and relationships from unstructured text.
