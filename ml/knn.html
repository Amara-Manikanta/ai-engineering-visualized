Algorithms

K-Nearest Neighbors (KNN)

An instance-based learning algorithm that memorizes training data and makes predictions by comparing new data points to stored instances.

Key Concepts of KNN

  • Instance-Based Learning: KNN doesn't explicitly train a mathematical model. It simply stores the training dataset and performs computation at prediction time.
  • Parameter K: The number of nearest neighbors to look at. A smaller K can be noisy and susceptible to outliers, while a larger K provides a smoother decision boundary.

Distance Metrics

Euclidean Distance

Straight-line distance between two points.
√[ Σ(xᵢ - yᵢ)² ]

Manhattan Distance

Sum of absolute differences across dimensions.
Σ|xᵢ - yᵢ|

Minkowski Distance

A generalized form of Euclidean and Manhattan distances parameterized by p.

How KNN Works

For Classification

  1. Store training data.
  2. For a new point, calculate distance to all training points.
  3. Select the K closest points.
  4. Assign the most common class (majority vote) among those neighbors.

For Regression

  1. Store training data.
  2. For a new point, calculate distance to all training points.
  3. Select the K closest points.
  4. Predict the value as the average of the values of those neighbors.

Evaluation Metrics

Accuracy, Precision, Recall, and F1
  • Accuracy: Proportion of correctly classified instances. Best for balanced datasets. (Correct / Total)
  • Precision: Proportion of true positive instances out of all predicted positives. Important when false positives are costly. (TP / (TP + FP))
  • Recall (Sensitivity): Proportion of true positive instances out of all actual positives. Important when false negatives are costly. (TP / (TP + FN))
  • F1 Score: Harmonic mean of precision and recall. Best for imbalanced datasets.

ROC Curve & AUC

The ROC Curve plots True Positive Rate vs False Positive Rate at various thresholds. AUC (Area Under Curve) ranges from 0 to 1, with 1 indicating perfect performance.

Cross-Validation

Technique to evaluate model performance by partitioning data into subsets (e.g., K-Fold). Essential for assessing generalizability to unseen data.
